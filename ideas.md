- `cat`, `less`, `find`, `grep`, `awk`, `ssh`/`scp`, `dd`, `rsync`, ...

- terminal multiplexer
- modern version of GNU Screen
- https://github.com/tmux/tmux/wiki

- http://byobu.co/

## Ranger

- file explorer

- fuzzy search
- `find * -type f | fzf`
- see `locate` for a static indexer
- category: search

- https://github.com/junegunn/fzf
